What is SPL?

SPL looks to simplify the creation of HTML documents and make web development easier and hopefully more effective with little to no HTML experience.

Motivation

To create a programming language that simplifies the creation of HTML documents. Normally, HTML documents may have a lot of repetitive code, and may be messy. Our goal is to generate HTML code through simple commands.

Features

  • Command that initializes the HTML document
  • Commands that create HTML components
  • Commands that create Bootstrap components
  • Basic styling commands
  • Add content to HTML documents

Documentation



Commands Parameters Description
start filename Command that generates the html file
add filename, tag, tag_id Command that adds an html component or tag to the end of the html file
addinside target_id, tag, tag_id Command that adds an html component or tag inside another html component or tag
addafter target_id, tag, tag_id Command that adds an html component or tag after another html component or tag
addbefore target_id, tag, tag_id Command that adds an html component or tag before another html component or tag
put target_id, content Command that allows content
style target_id, style_option Command that initializes the styling options
Syntax


HTML Component Description
p Defines a paragraph
div Defines a section in a document
h1 to h6 Defines HTML headings
img Defines an image element
Syntax


Bootstrap Component Description
row Defines a row
columns Defines a column: sm (for tablets), md (for desktops), lg (for larger desktops)
signup Defines a signup form
jumbotron Defines a jumbotron, a big box for calling extra attention to some special content or information
navbar Defines a navigation bar at the top of the website
gallery Defines a gallery of 6 images

Language Sintax Examples

Start

Command to generate the html file


    
                    start "index"

Add


    
                    add div "div_id"

                    add p "paragraph_id"

                    add h1 "header_id"

Add Inside


    
                    addinside "target_id" div "div_id"

                    addinside "target_id" p "paragraph_id"

                    addinside "target_id" h1 "header_id"

Add After


    
                     addafter "target_id" div "div_id"

                     addafter "target_id" p "paragraph_id"

                     addafter "target_id" h1 "header_id"

Add Before


    
                     addbefore "target_id" div "div_id"

                     addbefore "target_id" p "paragraph_id"

                     addbefore "target_id" h1 "header_id"

Put


    
                     put "target_id" "content"

Images

    
                    add img "image_id" "image_path"

                    addinside "target_id" img "image_id" "image_path"

                    addafter "target_id" img "image_id" "image_path"

                    addbefore "target_id" img "image_id" "image_path"
